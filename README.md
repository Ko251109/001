# yaque ブログ Obsidian Vault

> **目的**: 野球(yaque)ブログの文体を再現するAIエージェントのナレッジベース

---

## このVaultについて

このVaultは [野球(yaque)のblog](https://blog.livedoor.jp/ongaq89-yaque/) の全記事209本を解析して作成されました。
AIエージェントがこのVaultを参照することで、筆者の文体・語彙・構成パターンを忠実に再現したブログ記事を自動生成できます。

## エージェントの使い方

### 方法1：Obsidian + AIプラグイン
1. このVaultをObsidianで開く
2. `00_エージェント設定/AGENT_PROMPT.md` の内容をAIプラグインのシステムプロンプトに設定
3. 記事を書きたいライブ情報（アーティスト名、会場、日付、セットリスト等）を入力
4. AIが文体を再現した記事を生成

### 方法2：ChatGPT / Claude等のLLMに直接貼り付け
1. `00_エージェント設定/AGENT_PROMPT.md` をシステムプロンプトとして設定
2. `01_記事アーカイブ/ライブレポート/` から最新の記事を3〜5本コンテキストとして追加
3. ライブ情報を入力して記事生成を依頼

### 方法3：RAG（Retrieval-Augmented Generation）
1. 全記事をベクトルDBに登録
2. 類似記事を検索してコンテキストに追加
3. `AGENT_PROMPT.md` のシステムプロンプトと組み合わせて生成
