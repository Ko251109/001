# ライブの話(Spangle call Lilli line 4/27 六本木EX THEATER)

> **日付**: 2019年04月28日
> **URL**: [http://blog.livedoor.jp/ongaq89-yaque/archives/55226662.html](http://blog.livedoor.jp/ongaq89-yaque/archives/55226662.html)

Spangle call Lilli lineの20周年ライブに行ってきました。
最新アルバムのリード曲「red」からスタートし、序盤は「Dreams Never End」からの曲が続きます。普段はアルバムリリース後のライブでも最新曲を演奏しなかったりするらしいので貴重なのかも。
中盤には「dreamer」「azure」「B」などを披露。
「mio」は最新アルバムの曲ですが、SCLLらしさが詰まった新たな代表曲という感じがします。
ラストは色鮮やかなライティングの中「eye」で華々しく本編が締められました。
アンコールではグッズのTシャツを着て登場。Tシャツは黒地に白でdreams never endと書かれたものと白地に黒で「give each other space」と書かれたものの2種類があり後者が良く売れているそうです。このメッセージこそSCLLのすべてを物語っているように思えます。
アンコール1曲目は名曲「nano」。初めてこの曲を聴いた時の事を思い出しながら感慨深い気持ちで聴いていました。
続いて披露された「E」も良かったですね。
デビュー当時に経験したゲスト0ライブの記憶からライブ嫌いになっていたというSCLL。20年かかってEX THEATERを埋められて嬉しいとMCで語っていました。
また20周年といってもライブ活動としては正味2ヶ月くらいだとも。自分も彼らのライブを観たのはサマソニに出演した時一回きりなのでもっと活動してほしい反面、SCLLは4年に1回くらいのペースでも良いのでこの適度な距離感温度感で音楽を続けていってほしいなとも思います。
