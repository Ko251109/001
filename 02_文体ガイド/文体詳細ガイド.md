## 文の長さの分布

| 文の長さ | 割合 | 特徴 |
|---|---|---|
| 短文（〜20字） | 25% | 感想・印象を端的に |
| 中文（21〜50字） | 38% | 最多。情報と感想のバランス |
| 長文（51字〜） | 35% | 演奏描写・状況説明 |

**ポイント**: 中文が主体で、長文と短文を交互に使いリズムを作る。

## 文末表現の頻度（実測）

| 文末表現 | 特徴 |
|---|---|
| 〜ました | 最多。事実の記述に |
| 〜でした | 過去の状況・印象 |
| 〜ます | 現在進行・描写 |
| 〜です | 説明・定義 |
| 〜と思います | 主観的意見 |
| 〜と感じました | 感情の表現 |
| 〜という印象があります | 柔らかい評価 |
| 〜ようでした | 推量・観察 |
| 〜ではないかと思います | 控えめな断定 |

```

カーネーションのライブに行ってきました。
```

## 締め方パターン分析

### パターンA：余韻・感想でまとめる（最多）
```
〜という印象が残りました。
〜という余韻のまま公演は締めくくられました。
本当に行って良かったと思います。
```

### パターンB：次への期待で締める
```
次のツアーが今から楽しみです。
来年も精力的に活動していくことが伝わってきました。
また頑張って次のツアーのチケット取りたいと思ってしまいます。
```

### パターンC：アーティストへの敬意で締める
```
カーネーションは偉大だ。
こうやって脈々と繋がっていくのが良い音楽なのだなと、改めて実感した次第です。
```

## ライブ描写の特徴的表現集（実際の記事より抜粋）

### 開演・演出
- 「開演時間になると、舞台袖からゆっくりと4人が登場」
- 「オープニングは「〇〇」から」
- 「音響、照明をほぼ落とした暗闇からの演出」
- 「ステージが一度完全に暗転し」
- 「アウトロでは銅鑼がステージ中央に運び込まれ、最後に豪快に打ち鳴らして曲が終わります」

### 演奏描写
- 「演奏からは解放感が伝わってきます」
- 「Bメロからサビに至る流れが本当に心地良く」
- 「小編成で披露」
- 「フル編成のバンドサウンド」
- 「弾き語りのコーナー」

### 会場・観客
- 「会場の熱量が一段階上がったのをはっきりと感じました」
- 「フロア全体が高揚感に包まれます」
- 「会場全体が同じリズムでまとまり」
- 「観客の大合唱」
- 「一体感が生まれやすい」

### セットリスト言及
- 「序盤は〜など、〜な楽曲が続きます」
- 「中盤では〜」
- 「終盤は人気曲を連発」
- 「アンコールでは〜」
- 「定番曲と新曲を両立させた」

### MC描写
- 「MCでは、〜の話題もありました」
- 「MCは〜が中心で、普段着の〇〇らしい運び」
- 「〜と伝えて始まった「〇〇」」
- 「メンバーの発言を聴き」

## 感情表現の対応表

| 感情 | 直接表現（避ける） | yaque流の表現（使う） |
|---|---|---|
| 感動 | 感動した、泣いた | 「余韻のまま公演は締めくくられました」「胸に刺さります」 |
| 興奮 | 興奮した、テンション上がった | 「会場の熱量が一段階上がったのをはっきりと感じました」 |
| 楽しかった | 最高だった、楽しすぎた | 「非常に密度の濃い時間でした」「本当に行って良かったと思います」 |
| 期待 | 楽しみすぎる！ | 「今から楽しみで仕方ありません」「今から楽しみです」 |
| 寂しさ | 終わってほしくなかった | 「もうすぐ終わってしまうのかという寂しさもよぎります」 |
| 驚き | びっくりした、ヤバい | 「衝撃を受けました」「思わず〜してしまいました」 |

## 語彙の使い分け

### 「ライブ」の言い換え
- 公演、ワンマン、ツアー、ステージ、パフォーマンス

### 「演奏する」の言い換え
- 披露する、奏でる、響かせる、演じる

### 「盛り上がる」の言い換え
- 熱量が上がる、高揚感に包まれる、一体感が生まれる

---

## 絶対に使わない表現

- ヤバい、ヤバすぎる
- 神ライブ、神曲
- 最高すぎる、最高傑作（控えめに使う）
- 〜！！（感嘆符の連続）
- 草、笑（ネットスラング）
- 〜だよね、〜じゃん（タメ口）
