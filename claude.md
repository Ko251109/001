## 文体の核心ルール（必ず守ること）

### 1. 書き出しパターン
最も多いのは以下の形式：
```
〇〇のライブに行ってきました。
〇〇で開催された、〇〇の来日公演を観てきました。
〇〇のワンマンライブに行ってきました。
```
**絶対に避けること**: キャッチーすぎる書き出し、感嘆符から始まる文、「〜でした！」のような過剰な感情表現。

### 2. 文末表現
- **基本**: 「〜ました」「〜でした」「〜ます」「〜です」
- **感想**: 「〜と思います」「〜と感じました」「〜という印象があります」「〜ようでした」
- **禁止**: 「〜だ！」「〜すごい！」「ヤバい」「神ライブ」「最高すぎる」

### 3. 一人称の使い方
- **主語は省略する**（「私は」「僕は」を使わない）
- 「行ってきました」「感じました」「思います」のように主語なしで書く

### 4. アーティストへの言及
- ソロアーティスト: 「〇〇さん」（例：TOMOOさん）
- バンド: 敬称なし（例：カーネーション、GRAPEVINE）
- 初出時は正式名称、2回目以降は略称可

### 5. 曲名の表記
- 「」（鍵括弧）で囲む: 例「高台」「餃子」
- 英語曲名もそのまま: 例「Lip Noise」「DEAR MYSTERIES」

### 6. 感情表現のルール
**直接的な感情語は控えめに**。代わりに描写で感情を伝える。

| 避けるべき表現 | 代わりに使う表現 |
|---|---|
| 「感動した」 | 「余韻のまま公演は締めくくられました」 |
| 「最高だった」 | 「非常に密度の濃い時間でした」 |
| 「泣いた」 | 「胸に刺さります」 |
| 「興奮した」 | 「会場の熱量が一段階上がったのをはっきりと感じました」 |

**語尾に感想を無理やり結びつけない**。事実の描写や状況説明の文は、感想で締めず、事実のまま終わらせる。

| 避けるべき例 | 修正例 |
|---|---|
| 「18:00からほぼ遅延なしのスタートというのも気持ちよかった」 | 「18:00からほぼ遅延なしのスタートでした」 |
| 「即興はそれはそれで面白かったです」 | 削除、または前文に統合 |
| 「鉄板曲ももちろん現在進行形で輝いています」 | 「鉄板曲ももちろん演奏されています」 |

**使用しない表現**（筆者の好みに合わないと明示されたもの）:

| 避ける表現 | 理由・代替 |
|---|---|
| 「かなり久しぶりの再会でした」 | 感慨を無理に言語化している。事実として「一度だけ観たことがあったように思います」で終わらせる |
| 「空気を切り裂くようなギター」 | 比喩が大げさ。使わない |
| 「今日という日が迎えられて乗っかり得だなと」 | MCの引用内であっても、この語感は使わない。該当箇所を省略するか前文で止める |
| 「フロアの空気が変わりました」 | 常套句的で嘘くさい。雰囲気の変化を主観的に断言しない。客観的な事実（照明が変わった、演奏が静まった等）で代替するか省略する |
| 「〜という構成でした」「〜という形でした」 | 機械的・説明的で不自然。「〜が並びました」「〜が続きました」など具体的な動詞で書き直す |

### 7. AIっぽさの排除

既存記事（特に参照すべき代表的な記事）を参照し、以下を意識してAI生成文特有のパターンを避けること。

- **均質な文長を避ける**: 短文と長文を自然に混在させる
- **説明的すぎる構造を避ける**: 「〜という点で」「〜という意味で」のような解説口調を使わない
- **接続詞の多用を避ける**: 「また」「そして」「さらに」を連続して使わない
- **総括・まとめ文の型を使わない**: 「ユニットの形が異なっても〜一本の線が通っている」「またいつか〜を期待したい」のような感情の乗っていない締め方をしない

## 締め・総括の書き方ルール

総括・締めの段落は以下の3点を軸に構成すること。情報が不足している場合は執筆前に筆者にインタビューして確認する。

1. **このライブで感じた体験・感覚**: 会場や演奏を通じて実際に感じたことを具体的に
2. **これまでのライブとの相違点**: 同アーティストの過去公演と比べて異なった点
3. **他のライブにはなかった固有の要素**: そのライブ特有の演出・選曲・雰囲気

**禁止パターン**:
- 感情の乗っていない総括（「〜という印象がありました。またいつか〜を期待したいと思います。」のような型）
- 文字数を稼ぐための抽象的まとめ文
- 「一本の線が通っている」「つながりを感じました」のような感慨を無理に言語化した締め

```
[書き出し]
〇〇のライブに行ってきました。（1〜2文で文脈・背景を補足）

[会場・状況の説明]
今回の公演は〜。会場は〜。

[本編の流れ]
オープニングは「〇〇」から。（セットリストの流れを段落ごとに追う）
序盤は〜。
中盤では〜。
終盤は〜。

[MCや演出の描写]
MCでは〜。

[クライマックス]
〜で会場の熱量が一段階上がりました。

[アンコール]
アンコールでは〜。

[締め]
〜という印象が残りました。（次への期待や余韻で締める）
```
